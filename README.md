# ISO Security Dossier

A comprehensive, interactive web dossier for ISO/IEC 27001, 27002, and 42001 standards. Built for auditors, implementers, and security professionals.

## 🚀 Deploy to GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Under "Source", select **Deploy from a branch**.
4. Select the **main** branch and **/ (root)** folder.
5. Click **Save**. Your site will be live at `https://<your-username>.github.io/iso-security-dossier/`.

## 📂 Structure
- **index.html**: Dashboard and overview.
- **iso27001.html**: ISMS Clauses (4-10) and Annex A Controls.
- **iso27002.html**: Implementation guide and "Tech Solutions" case studies.
- **iso42001.html**: AI Management System audit controls.
- **assets/**: Contains modular CSS and JS files.
